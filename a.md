```uml
@startuml
ユーザー -> Webシステム :会員登録
Webシステム -> DBシステム :会員登録
DBシステム -> DBシステム :登録処理
Webシステム <- DBシステム :登録結果

alt 登録成功
 Webシステム -> ユーザー : 登録メッセージを表示
else 登録失敗
 Webシステム -> ユーザー : エラーを表示
 end
end

@startuml
 ユーザー -> Webシステム :ログイン
Webシステム -> DBシステム :ログイン
DBシステム -> DBシステム :ログイン処理
Webシステム <- DBシステム :ログイン結果

alt ログイン成功
 Webシステム -> ユーザー : 会員様を表示
else 登録失敗
 Webシステム -> ユーザー : 会員登録を進める
 end

@enduml
```
